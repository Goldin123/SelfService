***This is a help guide for using the application.***

A. Application first time use. (Creating the local DB)

1. Please double click folder 'DataCenter'.
2. Right click and build project 'SelfServiceSQLDB'.
3. Right click publish project 'SelfServiceSQLDB'.
4. Next to  option 'Target database connection' click on 'Edit'.
5. Click 'Show Connection Properties'.
6. Enter Server Name '(localdb)\ProjectsV13'.
7. Use Windows Authentication.
8. Select Database Name 'SelfServiceSQLDB'.
9. Click 'Test Connection' then 'Ok'.
10. Click on 'Publish'.
11. Wait for the publish process to finish.
12. Rebuild the whole solution.
130 Run the solution.

B. Application normal use.

1. Once you have completed the above you can just run the application as normal without re-creating the localDB
